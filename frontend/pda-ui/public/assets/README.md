# Assets Directory

Place the Solana logo image file here.

## Solana Logo

1. Save your Solana logo image file as `solana-logo.png` or `solana-logo.svg` in this directory.
2. The component will automatically use the image file if available.
3. If the image file is not found, it will fall back to the SVG version.

**Supported formats:**
- PNG (recommended): `solana-logo.png`
- SVG: `solana-logo.svg`
- JPG: `solana-logo.jpg`

The component looks for `/assets/solana-logo.png` by default.
